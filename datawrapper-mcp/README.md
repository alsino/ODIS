# Datawrapper MCP Server

*(Planned)* MCP server for creating data visualizations using the Datawrapper API.

## Purpose

This server will enable automatic chart generation from Berlin open data by:
- Accepting data in tabular format
- Creating charts via the Datawrapper API
- Returning chart URLs or embed codes
- Supporting common chart types (bar, line, scatter, etc.)

## Status

Not yet implemented.
